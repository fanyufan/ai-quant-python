MACD策略回测代码逻辑梳理

一、代码概述

文件名称：2-macd_strategy_2025.py

功能描述：基于MACD指标的交易策略回测系统，针对寒武纪股票（688256.SH）进行2025年全年的策略回测。

策略类型：技术指标策略 - MACD金叉死叉策略

回测方式：使用本地CSV数据进行历史回测

二、策略参数配置

参数名称 参数值 说明
STOCK_CODE 688256.SH 寒武纪股票代码
STOCK_NAME 寒武纪 股票名称
SHORT_PERIOD 12 MACD快线周期（EMA12）
LONG_PERIOD 26 MACD慢线周期（EMA26）
SIGNAL_PERIOD 9 MACD信号线周期（DEA）
START_DATE 2025-01-01 回测开始日期
END_DATE 2025-12-31 回测结束日期
INIT_CASH 1,000,000.0 初始资金（100万元）
LOT_SIZE 100 最小交易单位（1手=100股）
COMMISSION_RATE 0.0003 手续费率（万分之三，买卖双向收取）

三、核心策略逻辑

MACD金叉死叉策略

  • 买入信号（金叉）：当DIF线上穿DEA线时，满仓买入
  • 卖出信号（死叉）：当DIF线下穿DEA线时，清仓卖出
  • 持仓管理：全仓买入和全仓卖出，不进行部分仓位管理

MACD指标计算逻辑：

# MACD指标计算公式 # 1. 计算快线EMA(12)和慢线EMA(26) EMA_12 = ewm(close, span=12) EMA_26 = ewm(close, span=26) # 2. 计算DIF线（快线 - 慢线） DIF = EMA_12 - EMA_26 # 3. 计算DEA线（DIF的9日EMA） DEA = ewm(DIF, span=9) # 4. 计算MACD柱状图 MACD柱 = (DIF - DEA) × 2

交易信号判断：

# 金叉判断：DIF上穿DEA 金叉条件 = (DIF[i-1] ≤ DEA[i-1]) and (DIF[i] > DEA[i]) # 死叉判断：DIF下穿DEA 死叉条件 = (DIF[i-1] ≥ DEA[i-1]) and (DIF[i] < DEA[i])

四、核心函数说明

  • load_stock_data(data_file)

    功能：从CSV文件加载股票历史数据

    输入：CSV文件路径

    输出：包含日期和收盘价的DataFrame

    处理：读取CSV → 转换日期格式 → 按日期排序 → 返回数据

  • get_MACD(close, short, long, m)

    功能：计算MACD指标（DIF、DEA、MACD柱）

    输入：收盘价序列、快线周期、慢线周期、信号线周期

    输出：DIF数组、DEA数组、MACD柱数组

    算法：使用pandas的ewm函数计算指数移动平均线

  • calculate_max_drawdown(nav_series)

    功能：计算最大回撤

    输入：净值序列（pandas Series）

    输出：最大回撤值（负数）

    计算逻辑：回撤 = (当前净值 / 历史最高净值) - 1，取最小值

  • calculate_total_return(nav_series)

    功能：计算总收益率

    输入：净值序列

    输出：总收益率（小数形式）

    计算逻辑：总收益率 = (期末净值 / 期初净值) - 1

  • plot_strategy_results(...)

    功能：绘制策略回测结果图表

    输出图表：

    • 子图1：股价走势与买卖点标记
    • 子图2：MACD指标（DIF线、DEA线、MACD柱）
    • 子图3：资金曲线（净值变化）

    保存：图表保存为PNG文件到outputs目录

  • macd_strategy_backtest(data_file)

    功能：MACD策略回测主函数

五、回测流程详解

步骤1：加载历史数据

从CSV文件读取股票数据，包括日期和收盘价，并进行数据清洗和排序

步骤2：计算MACD指标

使用全部历史数据计算MACD指标（DIF、DEA、MACD柱），确保有足够的历史数据用于计算

注意：使用全部历史数据计算MACD，但只对回测区间（2025年）进行交易回测
步骤3：过滤回测区间数据

从全部数据中筛选出回测区间（2025-01-01至2025-12-31）的数据

步骤4：初始化回测状态

初始化现金、持仓、净值等变量

  • 初始现金 = 1,000,000元
  • 初始持仓 = 0股
  • 创建仓位和净值数组
步骤5：逐日回测循环

对回测区间内的每个交易日进行以下操作：

  1. 判断MACD信号：
    • 检查是否出现金叉（DIF上穿DEA）
    • 检查是否出现死叉（DIF下穿DEA）
  2. 执行买入操作（金叉且空仓）：
    • 计算可用资金（考虑手续费）
    • 计算可买入股数（向下取整到100的整数倍）
    • 计算买入金额和手续费
    • 更新现金和持仓
    • 记录交易信息
  3. 执行卖出操作（死叉且持仓）：
    • 计算卖出金额
    • 扣除手续费
    • 更新现金和持仓（清仓）
    • 记录交易信息
  4. 计算当日净值：
    • 净值 = 现金 + 持仓市值
    • 仓位比例 = 持仓市值 / 净值
步骤6：计算回测指标

计算策略表现指标：

  • 总收益率
  • 最大回撤
  • 交易次数
  • 期末净值和现金
步骤7：保存结果

将回测结果保存到文件：

  • 净值曲线CSV文件
  • 交易记录CSV文件
  • 汇总报告TXT文件
步骤8：绘制策略图表

生成可视化图表，包括股价走势、MACD指标和资金曲线

六、交易执行细节

6.1 买入逻辑

# 买入条件：金叉且当前空仓 if 金叉 and shares == 0: # 1. 计算可用资金（扣除手续费） 可用资金 = 现金 / (1 + 手续费率) # 2. 计算可买入股数（向下取整到100的整数倍） 最大股数 = int(可用资金 / 当前价格 / 100) × 100 # 3. 计算买入成本和手续费 买入金额 = 最大股数 × 当前价格 手续费 = 买入金额 × 手续费率 总成本 = 买入金额 + 手续费 # 4. 执行买入 if 总成本 ≤ 现金: 持仓 = 最大股数 现金 -= 总成本 记录交易信息

6.2 卖出逻辑

# 卖出条件：死叉且当前持仓 if 死叉 and shares > 0: # 1. 计算卖出金额和手续费 卖出金额 = 持仓 × 当前价格 手续费 = 卖出金额 × 手续费率 净收入 = 卖出金额 - 手续费 # 2. 执行卖出（清仓） 现金 += 净收入 持仓 = 0 记录交易信息

6.3 手续费计算

手续费率：0.0003（万分之三）

收取方式：买入和卖出双向收取

计算公式：

  • 买入手续费 = 买入金额 × 0.0003
  • 卖出手续费 = 卖出金额 × 0.0003

八、关键指标说明

8.1 总收益率

总收益率 = (期末净值 / 期初净值) - 1

衡量策略在整个回测期间的收益表现

8.2 最大回撤

回撤 = (当前净值 / 历史最高净值) - 1 最大回撤 = min(所有回撤值)

衡量策略的风险水平，最大回撤越小说明风险控制越好

8.3 交易次数

统计回测期间内执行的买入和卖出操作的总次数

注意事项：
  • 策略使用全仓买入和全仓卖出，不进行仓位管理
  • 交易必须满足最小交易单位（100股）的要求
  • 手续费会影响实际收益，已纳入计算
  • MACD指标需要足够的历史数据才能准确计算
